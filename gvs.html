Island Reef IoT Integrated Situation Monitoring and Control System

The project aims at designing an island reef situation-based display control system for multi-dimensional target monitoring and tracking. System use IoT platform to integrate multiple sensing device data and achieve multi-device cooperation, and also more accurately display the targets situation information surrounding the island, automatic track and lock objects.
